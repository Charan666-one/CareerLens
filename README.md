# CareerLens 🎯
 
> Graph-powered career intelligence for students targeting their first tech role.
 
## What it does
- Parses your resume with section-aware NLP
- Matches you to jobs using a hybrid scorer (semantic + graph proximity + market demand)
- Detects skill gaps using BFS traversal on a prerequisite knowledge graph
- Generates a sequenced learning roadmap via topological sort
- Shows exactly why each job was recommended (explainable AI)
 
## Tech Stack
- **Backend:** FastAPI, PostgreSQL, SQLAlchemy, SpaCy, Scikit-learn, NetworkX
- **Frontend:** React 18, TypeScript, TailwindCSS, Recharts, D3.js
- **Infrastructure:** Docker Compose, Railway (deployment)
